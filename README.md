## Estrutura do portfólio

```
portfolio/
├── index.html       # SPA estática — zero frameworks, zero build step
├── css/
│   └── style.css    # Design system com CSS custom properties
└── js/
    └── main.js      # IntersectionObserver, typed animation, mobile nav
```

O site é construído com **HTML5 semântico, CSS3 puro e JavaScript vanilla** — escolha intencional para demonstrar domínio dos fundamentos sem dependência de frameworks. Responsivo, acessível e pronto para deploy estático.
