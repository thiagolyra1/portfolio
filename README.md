# Portfolio — Thiago de Lyra Borges

Portfólio pessoal construído com HTML, CSS e JavaScript puros (sem build step).

## Tecnologias

- HTML5 semântico
- CSS3 com custom properties, Grid e Flexbox
- JavaScript vanilla (Intersection Observer, typed animation)
- Google Fonts: Inter + Fira Code

## Estrutura

```
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md
```

## Deploy

Compatível com **GitHub Pages**, **Netlify** e **Vercel** sem configuração adicional.

Para GitHub Pages: vá em `Settings → Pages → Deploy from branch → main / (root)`.
